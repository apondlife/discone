pondlife: discone (a videogame)

the first disc in the pondlife disc series; a run & jump & talk 'em-up. visit our wiki for more info!

controls

wasd / left analog stick: move body
ijkl / right analog stick: move camera
space / south: jump / talk
u / west / east: close eyes (imagine flower)
o / north: return to flower
enter: toggle menu

note

even if you think you're stuck, you might not be! it might actually be an interesting situation to try getting yourself out of. if you're really stuck, you might try holding "o". if you're really really stuck, you can open the menu and "disconnect" to restart the game (and connect again, if you were playing online).
